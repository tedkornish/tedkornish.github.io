Recent small personal project: jgrep, a tool for querying JSON-formatted logs using English-like syntax. Written in OCaml, it's basically a compiler which maps English into jq. Treating natural language querying like a rule-based compiler problem results in a small codebase (<300LOC) and solid performance (the actual work is being offloaded to a battle-tested tool!), if slightly inflexible syntax. A fun hack; OCaml is a delightful language.
